Club de danse sportive CDSM

Année : 2022

Projet réalisé en 1ère année

Le projet

L’association InterDance qui se trouve à Sainte-Colombe 69560 n’a pas de site web pour le club de danse CDSM Vienne. Suite à cela, notre équipe va mettre en place un site web du club avec les informations suivantes : sur les anciens sportifs, ainsi que les récents ; sur les professeurs ; les détails sur les différents types de danses ; les compétitions nationales ou internationales .

Objectif du projet

Le but de notre projet est de créer le site web qui permet d’informer sur le club de danse, ainsi que la danse sportive, les professeurs et les danseurs. Il permettra aux publiques de connaître la danse sportive et tout l’historique du club.

    Ged’Imagination

    Année : 2022-2023

    Projet réalisé en 2ème année

    Ce projet a été réalisé de Novembre 2022 à ??? 2023. Ce projet avait un vrai contexte professionnel et il a été découpé en différents sprints pour pouvoir avancer petit à petit méthodiquement.

    Contexte

    La société NEGOMAT vient tout juste d’entrer dans le réseau d’adhérents de GEDIMAT. Pour fêter cet événement, NEGOMAT a décidé d’organiser un jeu-concours nommé «GEDIMA’GINATION »

    Mission

    Mettre en place le jeu-concours en créant une application Web, mobile et une API.

    Principe du jeu

    Le jeu est ouvert aux clients de NEGOMAT. Pour participer, il suffit de s’inscrire et poster une photo d’une réalisation faite à partir de matériaux achetés chez NEGOMAT. Exemples : aménagement d’un extérieur, rénovation d’une salle de bain, réalisation d’une terrasse en bois,... Dans un deuxième temps, les clients de passage dans le magasin voteront pour leurs 3 réalisations préférées. Les photos des réalisations seront consultables sur un écran TV. Les clients utiliseront une borne tablette installée dans le magasin pour saisir leurs votes. À la fin du concours, seront désignés les 3 gagnants. Ceux-ci seront récompensés par des bons d’achat.

    Environnement de développement
    • EDI : Visual Studio Code + Android Studio
    • BDD : MySQL + SQL Lite
    • Outils : GitHub + Postman + Trello
    Bilan

    Ce projet m'a permis de toucher à plusieurs technologies, ce qui m'a aidé à acquérir et développer de nombreuses compétences.

    Site 1001 Visites

    Année : 2023

    Projet réalisé en stage de 2ème année

    Contexte

    J'ai effectué les tests fonctionnel sur le site 1001 Visites qui un site qui repértorie des visites 360° (d'entreprise, d'ecole, de casino ect...), afin de simuler l'input utilisateur. J'ai travaillé avec Selenium Web Driver qui est un outil open source pour les tests automatisés d'applications web sur plusieurs navigateurs. Il fournit des fonctions pour naviguer vers les pages web, simuler l'input utilisateur, l’exécution de JavaScript, etc...

    J'ai effectué des tests qui vérifient notamment par exemple que quand l'utilisateur sélectionne dans la catégorie département, le département de l'Isère, cela vérifie qu'il n'y est pas une visite d'un autre département que l'Isère.

    Mission du Stage :
  • Objectif : Réalisation de tests fonctionnel afin de vérifier d'éventuel problème sur le site à corriger
  • Environnement Technologique : JavaScript, Sélénium, Windows PowerShell
  • Environnement de développement : Visual Studio Code
    • Bilan Personnel :
  • Ce stage m’a appris la création de tests fonctionnel en Javascript avec Sélénium.
